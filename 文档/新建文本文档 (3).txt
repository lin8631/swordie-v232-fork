- **`npm install && npm run build`** — 当前环境没有 Node.js，无法验证前端构建

### 下一步你需要做
```bash
# 1. 安装前端依赖并构建
cd gms-ui
npm install
npm run build

# 2. 编译 Java 后端
cd ..
mvn compile

# 3. 启动服务，访问 http://localhost:3000
```

前后端启动后，登录流程：前端 POST `/api/auth/v1/login` → 后端验证密码 → 返回 token → 前端存储 token → 后续请求带 `Authorization: Bearer <token>` 头。
